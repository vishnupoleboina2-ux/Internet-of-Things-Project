# Internet-of-Things-Project

## IoT-Based Temperature and Humidity Monitoring System

### Project Description
This project demonstrates an Internet of Things (IoT) based Temperature and Humidity Monitoring System using Arduino Uno, DHT11 Sensor, and ESP8266 Wi-Fi Module. The system collects environmental data and displays it through the Serial Monitor. It helps understand IoT communication and sensor interfacing.

## Components Used
- Arduino Uno
- DHT11 Temperature & Humidity Sensor
- ESP8266 Wi-Fi Module
- Breadboard
- Jumper Wires
- USB Cable

## Features
- Measures Temperature
- Measures Humidity
- Real-time Sensor Monitoring
- Easy to Build
- Beginner Friendly

## Working
1. The DHT11 sensor reads temperature and humidity.
2. Arduino processes the sensor data.
3. ESP8266 enables IoT communication.
4. The readings are displayed on the Serial Monitor.

## Applications
- Smart Home
- Weather Monitoring
- Agriculture
- Industrial Monitoring
